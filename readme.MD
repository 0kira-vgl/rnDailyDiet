## About the Project

**Daily Diet** is a mobile app designed to help users track their meals and maintain control over their diet. With this app, you can:

- Register meals,
- Check if they're within or outside the diet,
- Track your progress with statistics,
- Edit or delete meals as necessary.

The app boasts an intuitive and visually appealing design, with color coding to highlight meals that are within (green ğŸŸ¢) or outside (red ğŸ”´) the diet. Additionally, it provides motivational feedback to encourage users.
